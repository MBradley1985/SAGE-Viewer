# Library

The **Library** tab lists all screenshots and movies captured in the current session.

## Browsing

Each row shows the filename, format, and a thumbnail or icon. Rows are sorted by capture time.

## Opening files

**Double-click** any row to open the file as a **floating card** over the viewport. The card is:

- Movable — drag the title bar
- Resizable — drag the bottom-right corner
- Semi-transparent background so the 3D scene shows through

Multiple items can be open simultaneously; each new card opens at a slightly offset position so they don't stack directly on top of each other.

## GIFs and movies

GIFs always start from frame 0 when opened or re-opened. Movies (MOV) open in the browser's native video player.

## Closing a card

Click the **×** in the card's title bar. This closes the floating card only — the file remains in the Library and on disk.

## Deleting files

Each row has a **delete** button (🗑) that removes the file from disk immediately. There is no undo. The row is removed from the Library list.
